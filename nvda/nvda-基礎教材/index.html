基礎教材 2.1 啟動 NVDA

  1. 通常 NVDA 會自己啟動
  2. 若沒有啟動可以按下 control + alt + n 來啟動 您會聽到油低到高的提示聲音
  3. 若 NVDA 正在運行中按下 control + alt + n 將重啟 NVDA. 會先聽到由高志低的提示生，這就是冠掉了 NVDA 接著才打開她
  4. control + alt + n 這表示您需要先按下 control不放開接著按下 alt一樣手不要放開接著再按下 n 然後說好了手放開
  5. 操作不方便的人可以開啟鄉年鑑 連按武嚇 shift 來啟動 此時，可以依序按下 control,alt,n 來開關 NVDA

案件位置的介紹

這裡說的是最常見的鍵盤布局

  • control 在最左下角
  • 往右一個則是 windows 案件
  • 載往右一個 則是 alt
  • 接著往右 最長的案件 他是 space
  • 在往右 是一顆 alt
  • alt 的右邊是 windows案件
  • 在往右 叫做快顯功能鍵 application
  • 在往右 則是 control

你會發現 control , win , alt 算是蠻對稱的
這也是一種人性化的設計，
不管事習慣用右手按或左手按都能達到目的。

基礎教材 2.2 NVDA 控制鍵

  • nvda件 主要用於與NVDA互動單案他是不起反映的，需要搭配其他的案件或者雙吉他 如讓NVDA讀出時間你可以按下 NVDA + F12
  • 哪一科案件是 NVDA Key 這取決於 NVDA 的鍵盤配置 若是桌上型電腦 insert 匯市您的 NVDA Key 而若是筆記型電腦 大小寫切換鍵是你的 NVDA Key

案件說明

大小寫切換鍵就在左下角的 control 往上屬兩個
往上屬第一顆是 shift 可以把他想成轉換的意思如小寫變成大寫 未選擇變成以選取等等
接著往上屬 就是大小寫切換案件
再往上 則是 tab 通常拿來跳轉項目或者是輸入 tab

基礎教材 2.3 再談 NVDA 控制鍵

  • 讀取時間很簡單 nvda + f12,而讀取日期同樣是按著 NVDA + double F12快按兩下
  • 像這樣快按兩下或三下的設計 在 NVDA 還蠻常見的 一定要跟著練習試試看
  • 原本大小寫切換間或 insert 是有它的功能但現在被 NVDA 占用，所以你要按兩下才會執行原本的功能

基礎教材 2.4 停止朗讀

  • 語音正在朗讀時可以按下 shift 暫停朗讀，在案一夏可以繼續朗讀
    • 在編輯文本區可會切換輸入法而停止朗讀
  • 語音朗讀時按下 control 可以停止朗讀
  • 暫停可以再繼續朗讀 停止則不行

指派案件只讓其中之一的 shift 切換輸入法 另一顆就可以拿來停止語音朗讀

win 11

  • win + i 打開設定

  • 輸入
  • 進階鍵盤設定
  • 輸入語言快速鍵
  • 切換語言 找到變更之後就可以改變了

基礎教材 2.5 輸入說明模式

  • 開關輸入說明模式 NVDA + 1(大鍵盤數字依) 分別為聽到 輸入說明 開 NVDA + 1 輸入說明關的提示因
  • 打開輸入說明模式後
    • 按下案件 nvda 會朗讀該案件
    • 若案的事 nvda 的功能案件會朗讀其功能
    • caps lock num lock insert 等 NVDA 功能鍵按兩下會告訴你原有的案件功能

基礎教材 2.6 使用點顯器來閱讀資訊

  • 語音朗讀的資訊能同樣的在點字顯示器上被獨到
  • 點顯氣空間有限 一些常用的電腦符號會被簡寫 也提供二級與三級的縮寫點字
  • 有點顯器的人 可以先去看看 點字資訊與語音自訓 有哪些差別

基礎教材 2.7 重新啟動與停止 NVDA

  • 啟動 NVDA 的快速鍵是?

nvda + control + n

  • 關閉 NVDA 可以按下 ctrl + q
    • 會彈出對話框 問你想做什麼 游標上下可以選擇項目 enter 可以執行功能
    • 若找到 結束 並按下 enter NVDA 就會被關閉
    • 若選到 重新啟動 按下 enter 以後 就相當於之前介紹的 nvda + ctrl + n 先關閉 nvda 後再啟動她
    • 重新啟動並停用附加元件 附加元件會改變 nvda 的行為與運作 主要是讓 NVDA 更好用才會開發此功能 但若有NVDA運作不順或異常的情形就可以利用功能檢查看看是否是 nvda-addons 的問題
    • 若不想看到此對話框 你可以在 NVDA功能表 - 偏好 - 設定 - 一般 - 結束 NVDA 時顯示結束選項(W) 勾選 Alt+w
      • 勾選 就是要顯示對話框 沒勾選 當然就是不顯示了

基礎教材 4 NVDA 基本設定

這章介紹了 利用 nvda 功能表對 nvda 做基礎的設定，
通過那些設定是為了讓使用者更好的利用 nvda 的功能。

基礎教材 4.1 NVDA 功能表

功能表 是功能的集合 通常以清單的形式展示
開始功能表與 nvda 功能表就是這樣的類型。

我們開 nvda 的功能表看看：nvda + n

  • 使用上下可以瀏覽清單內的項目
    • 有的清單可以再展開成為子功能表 瀏覽時若是有聽到子功能表的就是 可以用游標往誘惑 enter 展開 往左退到上一層的清單 點顯氣會顯示 submnu
    • 有的按下 enter 會打開對話框 組態設定檔(C)... c 就是這樣的 若有使用觸摸顯示器後面有(...)可以注意
    • 有的會執行功能如儲存組態(S) s
  • 在功能表內按下 esc 可以關閉他

基礎教材 4.2 NVDA 設定對話框

  • nvda功能表中的偏好裡頭的設定是個對話框
  • 利用游標可以瀏覽類別 記住別案 enter 因為一道哪個類別就自動選到哪個類別
  • 另外如果聽到 一般 21 之 1 代表有二十一個項目 目前在第一個項目上
  • 利用 tab / shift tab 可以瀏覽類別裡的設定項目
  • 在對話框中按下 esc 就會將對話框給關掉
  • 當你開啟了對話框 nvda 的功能表將被關掉

基礎教材 4.3 NVDA 的語音設定

接著我們來看看設定對話框的第二個項目 語音
其中第一個項目不能編輯 他是告訴你要利用哪個語音合成氣讀出文字
變更 按鈕 這邊若按下 enter / space 就可以去更換合成器
若在對話框非按鈕的地方按下 enter 預設試紙確認
若是在對話框中按下 esc 預設就是取消 不設定並放棄變更
套用 這裡是會將設定套用而不離開對話框
回到合成器，每個合成器都有不同的功能，有的可以變更朗讀腳色、語言等等。

基礎教材 4.4 探索語音合成器的對話框

打開語音合成氣的對話框
類別清單中選擇語音
按下 tab 找到變更的按鈕 enter
這張耀解紹的是如何操作下拉方塊，
是時尚這非常簡單，就是利用游標在項目內移動舊型，
另外一種是輸入自首。

在語音合成氣得對畫框中有三個夏拉方塊

  1. 語音合成器 選擇要用哪個合成器發音 espeak / onecore...
  2. 音訊輸出裝置 決定了 nvda 的聲音要從哪裡發出來 通常不用去設定他 若有多張音效卡的情形或接著藍芽的情況才需要調整
  3. 自動音量調整模式(快速鍵為 NVDA + shift + d) 當有其他的聲音輸出時如播方音樂 可以去調低非 nvda 的聲音，讓 nvda 的聲音更明顯

基礎教材 4.5 恢復到已儲存的設定

  • 若調整不符合您的心意，在還沒存檔的時候可以按下 nvda + control + r 恢復已儲存的設定
  • 結束 NVDA 時儲存組態(S)
    * 弱狀態是勾選 結束nvda時將自動儲存設定 當你重開之後設錯的依然是設定錯誤的
    * 弱狀態是沒勾選 重啟之後將回復到上一次儲存組態的時候

基礎教材 4.6 NVDA 的語音設定

選擇好了語音合成氣 你可以調整語音腳色 語音因高 速度 音量，這會讓你在聽讀的過程中有不同的體驗，值得來了解遺下。

基礎教材 5 儲存與開啟檔案

這一章的目標是要開始輸入文字與使用檔案，撰寫與編輯文字、儲存與開啟檔案，這些對所有使用者來說都是基本的操作。
NVDA 對於讀與寫的朗讀回饋有很多選項可以調整，認識這些已存在的選項與如何調整它們，有助於使用者更有效率的達成操作。

基礎教材 5.1 撰寫一個簡單的文件

  • 寫文章、寫信、寫code統一會用到的技能就是輸入文字
  • 再輸入的時候預設情況nvda 會將輸入文字朗讀出來讓你知道
  • 若打錯字可以按下到退件 backspace 當按下倒退健兒聽到文字時 該文字就被刪除了 如擬輸入了 帽子 按下到退件就是 子帽
  • 活動要麻煩跟著做一次喔 如果妳都沒試過將較難體會

基礎教材 5.2 使用方向鍵來瀏覽文字

  • 游標就是插入點
  • 在打字的時候游標會在字的右邊
  • 插入符號可以用方向建移動 當插入符號在行的中間，當你打字時就會插入在行的中間並將原本的文字往右移動
  • enter 可以拿來換行 與插入空白行
  • 左右移動游標 可以一次移動一個字元 且由標語字元是在同一個位置
  • 當游標往左移動時 效果與倒退見差不多 只是沒有刪除掉文字而已
  • 按下numpad(8) 可以朗讀該行 連按兩下 numpad(8) 可以一個字一個字的閱讀
  • nvda + 連按兩下某件或三下某見的情況是很常見的 如nvda + double[T] / nvda + threeble(T) 你可以試試看

基礎教材 5.3 存檔與關閉文件

  • ctrl + s 可以存檔
  • wordpad 預設的存檔路徑是 文件 我們先不選擇其他的儲存位置
  • 接著你需要輸入黨名 有一些符號不能使用 但數字、英文、中文都是沒問題的
  • 接著找到儲存按下 enter 就完成存檔了

基礎教材 5.4 開啟已儲存的文件

  • ctrl + o 可以打開開啟檔案的對話框 事實上這與存檔的對話框是一樣的
  • 打開對話框焦點會存在輸入黨名的位置 可以按 shift + tab 找到檔案名稱的清單 這裡就可以選擇檔案
  • 若焦點獨到 項目檢視 名稱 那你還要再按一次 shift + tab
  • 在檔案名稱的清單中 可以利用游標上下選擇檔案 and press enter to open file.
  • 閱讀文件的時候 也可以用游標上下建築行閱讀瀏覽。
  • hey 所以你在上個活動若有存檔你就不用再重做一次檔案了...

基礎教材 6 編輯文件

目標是學會瀏覽與編輯

基礎教材 6.1 閱讀文字

  • nvda + downarrow 可以從焦點處依序往下逐行閱讀 焦點會跟著移動
  • 利用 shift 可以暫停/播放語音 control 可以停止語音 不是每個語音合成氣都是用
  • 若想要逐行閱讀 只需利用游標上下就可以在文件或網頁逐行讀取
  • nvda + uparrow 可以朗讀當前行

基礎教材 6.2 更多瀏覽運用

  • 方向件左/右 可以個各自元的朗讀
  • 游標上/下 逐行閱讀
  • control + left/right 可以逐詞朗讀 英文是單字中文則是慈
  • control up/down 竹段閱讀 主要是看 enter 的換行符號而非電腦自動換行的結果
  • home 移道行手
  • end 移道行委
  • control + home 一道檔案開頭
  • control + end 一道檔案結尾

這些功能鍵通常在 wordpad , microsoft word , notepad 都是用

基礎教材 6.3 使用刪除鍵

  • 游標若在文字右邊 按下 backspace 就會將左邊的字給刪掉 同時 nvda 會朗讀刪除的文字
  • 若在編輯文字的時候按下 delete 刪掉的就是在定位的當下的那個字
  • 如果利用 ctrl + backspace / delete 那就像築慈瀏覽一樣會刪掉中文的一個詞或英文的一個單字

基礎教材 6.4 輸入字元的回饋設定

  • 朗讀字元開關 nvda + 2
  • 朗讀單字開關 nvda + 3
  • 朗讀字員若開 就像原本那樣 美輸入一個字 nvda 就會報讀 若關閉就不會報讀
  • 單字的開關 只對英文有效 英文每個單字都須以空格隔開 按下空格的時候 nvda 就會報讀出來 若關閉則不會 中文則沒有差
  • 這些設定可在 設定類別的鍵盤中找到

基礎教材 6.5 設定符號等級

  • nvda + p 可以調整符號等級
  • 當 nvda 朗讀時 決定朗讀出多少的標點符號
  • 等級依序是 沒有、一些、多數、全部
  • 這只在朗讀時有效，若是輸入文字，就不再此列
  • 同樣可以在語音中的符號等級做調整

基礎教材 6.6 讀出指令鍵

  • nvda + 4 可以讀出指令建 包刮了 tab 方向鍵 或著組合案件 ctrl + s
  • 在 wordpad 方向鍵不會唸 也是方便你閱讀
  • 開關指令見的設定在類別的鍵盤裡頭 讀出指令鍵(O)

基礎教材 6.7 選取文字

  • 選取文字以後我們可以來刪除、移動、複製編輯他們
  • 利用 shift 加上之前介紹的瀏覽按鍵就能將該字、慈、斷給選起來 control + shift + rightarrow 就是往右選一個單字或字詞
  • 請注意，若選取文字請固定一個方向 若反向就會變成部選取
  • 在選取的時候 nvda 惠告訴你選了那些字詞 並在後面追加以選取的提示 點顯器有不閃爍的 78點當作提示

基礎教材 6.8 沒選取文字

  • 選取的文字可以混用 shift + end , shift + downarrow , shift + rightarrow...
  • ctrl + a 可以全選
  • 選取都是同一個方向選擇 若是前進也就是往文件的後方移動 那就要按照這個方向前進
  • 若反向了 就是沒有選取某些文字 nvda 也會告訴你
  • nvda + shift + up 會朗讀所選內容
  • nvda + tab 除了朗讀全選內容還會朗讀焦點資訊
  • 選取文字後 請勿移動插入點 否則將取消選取

基礎教材 6.9 複製文字到剪貼簿

  • control + c 複製到剪貼簿
  • control + v 粘貼
  • nvda + c 朗讀剪貼簿內容

基礎教材 6.10 剪下文字

  • ctrl + x 可以剪下文字到剪貼簿裡頭 其他的操作方式就一樣啦
  • 粘貼方法一樣 ctrl + v

基礎教材 6.11 使用復原

復原就是回復到上一步的狀態 刪了一個字案移下復原 自救會回來
如 abcde 不小心刪掉了 e 按下復原 e 就會回來
復原的方式 ctrl + z

基礎教材 6.12 搜尋文字

  • ctrl + f 可以打開搜尋對話框
  • 輸入要搜尋的文字 按下 enter
  • 若有搜尋到焦點就會移過去 你要按下 esc 就可以看到搜尋的文字區段 此時文字是被選取的狀態
  • 若沒有搜尋到 會有個提示訊息跟你說
  • 想要順著搜尋 只需要按下 F3 同樣的 shift 轉換就會變成逆向搜尋
  • 搜尋框還有一些可調整的 可以去看看

基礎教材 6.13 本章回顧

瀏覽、nvda的閱讀命令、複製、粘貼、還原、剪下等等都必須熟悉，算是非常常見的命令。

基礎教材 7 文件格式

了解文件格式包刮了自行、字體、顏色等等，這邊會教大家如何取得這些資訊以便能夠調整他。

基礎教材 7.1 朗讀文字格式

  • 按下 nvda + f 就可以朗讀文字格式 字體、顏色、自行、大小
  • 按下 nvda + double[F] 會彈出對話框 你可以用游標讀取格式資訊 esc 可以關掉這個對話框
  • 選取文字後 利用 ctrl + b 可以將文字轉成粗體表示

基礎教材 7.2 當格式改變時自動朗讀

  • nvda + ctrl + d 可以打開文件格式設定框
  • 其中，讀出游標經過後的格式變化 (可能導致反應延遲)(G) ，這個選項若打勾當在整行朗讀的時候就會唸出文字的變化錫，
    如果沒有打勾，則只會在焦點經過的時候才會唸出來。

基礎教材 7.3 使用文字格式

有一些快速鍵能快速地調整格式資訊，且在大部分的城市都能夠使用:

使用方式:

  • 按下功能鍵 → 輸入文字 → 關閉功能
  • 另一種是先選取文字再按下功能鍵

功能鍵：

  • ctrl + b 粗體
  • ctrl + i 斜體
  • ctrl + u 文字家底線
  • ctrl + l 文字靠左對其
  • ctrl + r 靠又對奇文字
  • ctrl + e 文字致中
  • ctrl + j 文字左右對其
  • ctrl + shift + . 增大文字 no +1+1+1
  • ctrl + shift + , 減小文字

基礎教材 7.4 使用帶狀功能表

  • 功能表通常在標題列的下方 按下 alt 可以打開她
  • 左右選擇不同欄位
  • ctrl 左/右 可以瀏覽不同群組
  • tab 可以瀏覽項目
  • 在聽讀功能的最後 會寫出怎麼快速使用功能的 案件 季的因為它是用 , 隔開所以是一個一個地案

基礎教材 7.5 更多關於功能表

  • ctrl + f1 可以在畫面上決定是否隱藏功能表 對 nvda 沒差
  • 當功能表是隱藏的時候 按下 alt 焦點會停在檔案上
  • 若非隱藏的情況 焦點會停在你上次使用的欄位上
