[基礎教材] 2.1 啟動 NVDA

  1. 通常 NVDA 會自己啟動
  2. 若沒有啟動可以按下 control + alt + n 來啟動 您會聽到油低到高的提示聲音
  3. 若 NVDA 正在運行中按下 control + alt + n 將重啟 NVDA. 會先聽到由高志低的提示生，這就是冠掉了 NVDA 接著才打開她
  4. control + alt + n 這表示您需要先按下 control不放開接著按下 alt一樣手不要放開接著再按下 n 然後說好了手放開
  5. 操作不方便的人可以開啟鄉年鑑 連按武嚇 shift 來啟動 此時，可以依序按下 control,alt,n 來開關 NVDA

案件位置的介紹

這裡說的是最常見的鍵盤布局

  • control 在最左下角
  • 往右一個則是 windows 案件
  • 載往右一個 則是 alt
  • 接著往右 最長的案件 他是 space
  • 在往右 是一顆 alt
  • alt 的右邊是 windows案件
  • 在往右 叫做快顯功能鍵 application
  • 在往右 則是 control

你會發現 control , win , alt 算是蠻對稱的
這也是一種人性化的設計，
不管事習慣用右手按或左手按都能達到目的。

[基礎教材] 2.2 NVDA 控制鍵

  • nvda件 主要用於與NVDA互動單案他是不起反映的，需要搭配其他的案件或者雙吉他 如讓NVDA讀出時間你可以按下 NVDA + F12
  • 哪一科案件是 NVDA Key 這取決於 NVDA 的鍵盤配置 若是桌上型電腦 insert 匯市您的 NVDA Key 而若是筆記型電腦 大小寫切換鍵是你的 NVDA Key

案件說明

大小寫切換鍵就在左下角的 control 往上屬兩個
往上屬第一顆是 shift 可以把他想成轉換的意思如小寫變成大寫 未選擇變成以選取等等
接著往上屬 就是大小寫切換案件
再往上 則是 tab 通常拿來跳轉項目或者是輸入 tab

章節裡的活動請跟著練習看看

[基礎教材] 2.3 再談 NVDA 控制鍵

  • 讀取時間很簡單 nvda + f12,而讀取日期同樣是按著 NVDA + double F12快按兩下
  • 像這樣快按兩下或三下的設計 在 NVDA 還蠻常見的 一定要跟著練習試試看
  • 原本大小寫切換間或 insert 是有它的功能但現在被 NVDA 占用，所以你要按兩下才會執行原本的功能

[基礎教材] 2.4 停止朗讀

  • 語音正在朗讀時可以按下 shift 暫停朗讀，在案一夏可以繼續朗讀
    • 在編輯文本區可會切換輸入法而停止朗讀
  • 語音朗讀時按下 control 可以停止朗讀
  • 暫停可以再繼續朗讀 停止則不行

指派案件只讓其中之一的 shift 切換輸入法 另一顆就可以拿來停止語音朗讀

win 11

  • win + i 打開設定
  • 時間與語言
  • 輸入
  • 進階鍵盤設定
  • 輸入語言快速鍵
  • 切換語言 找到變更之後就可以改變了

[基礎教材] 2.5 輸入說明模式

  • 開關輸入說明模式 NVDA + 1(大鍵盤數字依) 分別為聽到 輸入說明 開 NVDA + 1 輸入說明關的提示因
  • 打開輸入說明模式後
    • 按下案件 nvda 會朗讀該案件
    • 若案的事 nvda 的功能案件會朗讀其功能
    • caps lock num lock insert 等 NVDA 功能鍵按兩下會告訴你原有的案件功能

[基礎教材] 2.6 使用點顯器來閱讀資訊

  • 語音朗讀的資訊能同樣的在點字顯示器上被獨到
  • 點顯氣空間有限 一些常用的電腦符號會被簡寫 也提供二級與三級的縮寫點字
  • 有點顯器的人 可以先去看看 點字資訊與語音自訓 有哪些差別

[基礎教材] 2.7 重新啟動與停止 NVDA

  • 啟動 NVDA 的快速鍵是?

nvda + control + n

  • 關閉 NVDA 可以按下 ctrl + q
    • 會彈出對話框 問你想做什麼 游標上下可以選擇項目 enter 可以執行功能
    • 若找到 結束 並按下 enter NVDA 就會被關閉
    • 若選到 重新啟動 按下 enter 以後 就相當於之前介紹的 nvda + ctrl + n 先關閉 nvda 後再啟動她
    • 重新啟動並停用附加元件 附加元件會改變 nvda 的行為與運作 主要是讓 NVDA 更好用才會開發此功能 但若有NVDA運作不順或異常的情形就可以利用功能檢查看看是否是 nvda-addons 的問題
    • 若不想看到此對話框 你可以在 NVDA功能表 - 偏好 - 設定 - 一般 - 結束 NVDA 時顯示結束選項(W) 勾選 Alt+w
      • 勾選 就是要顯示對話框 沒勾選 當然就是不顯示了

  2. tab 可以找到pinned tiles
  3. tab 所有城市區塊 按下 enter 且游標往下拉就可以看到所有城市

[基礎教材] 3.2 系統焦點

系統焦點是可直接與 nvda 互動的 通常直接按下 enter / space 就能與焦點項目互動
想要讀取焦點資訊可以按下 NVDA + tab 你可以打開功能表嘗試看看

在任何地方都可以試試看 桌面、開始功能表...

[基礎教材] 3.3 搜尋與開啟程式

  • 我們試著打開 wordpad
  • 按下 win 會停在搜尋的地方
  • 輸入 wordpad 載入路的過程中電腦就會將相關的選項列出來 所以也許不用打得很完整就會出現 wordpad
  • 想要打開城市 直接按下 enter 就可以

可以試著只打上 word 游標往下找找看有哪些城市

[基礎教材] 3.4 朗讀當前視窗的標題列

win + t 可以朗讀標題 與 nvda + tab 一樣在哪裡都是用 wordpad , 開始菜單 , 桌面

[基礎教材] 3.5 程式標題與焦點的差異

  • NVDA + t 是朗讀當前焦點所在的程式或開啟檔案的標題列資訊。
  • NVDA + tab 是朗讀當前焦點所在項目的名稱與類型。

向上一張一樣 在哪都可以按按看 聽聽兩者的區別

[基礎教材] 3.6 關閉 WordPad

  • 想要關閉城市 原則上只需要按下 alt + F4
  • 若再 wordpad 有編輯的行為按下 alt + F4 會跳出對話框問你要不要存檔 利用 tab 或者游標可以瀏覽項目
  • 我們選擇不要 程式就會關閉
  • 選擇取消 就會回到編輯畫面 相當於按下 ESC
  • 存檔我們暫且不提
