Win 設定環境變數

 windows  337  1 分鐘

我們使用 gui 的方式

Windows 11 22H2 (AMD64) build 22623.1095

設定環境變數

  • win + i 打開設定
  • 選擇系統
  • 接著 按下 tab 找到另一個清單找到 系統資訊 enter 另外你也可以在桌面找到畚箕按下 alt + enter
  • tab 找到進階 enter
  • tab 找到環境變數 enter
  • 這邊有兩個區塊 在上方的是使用者的環境變數 下方的是系統的環境變數 NVDA 會朗讀使用者或系統
  • 先找到 使用著的區域 游標可以上下移動看看有那些環境變數
  • 比較重要的環境變數是 path 我們將會在這裡掩飾
  • 游標找到 path 按下 tab 找到編輯 enter
    • 這邊會顯示 path 的值 這就是環境變數裡的內容
    • 若要新增路徑近來 按下 tab 找到 新增 enter 輸入路徑就可以
    • 若想要刪除找到該路徑按下 tab 找到刪除並按下 enter
    • 發現路徑寫錯了 就按下編輯 enter 然後寫路徑近來按下 enter
  • 當路徑編輯完成案 tab 找確定 enter 直到找不到確定就可以了

確認變數是否設定完成

win + r 打開輸入對話框
輸入 cmd 按下 ctrl + shift + enter 以管理員權限打開
輸入 path 檢查看看
