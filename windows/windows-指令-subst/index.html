Windows 指令 Subst

 2023.1.4  windows  549  2 分鐘

subst 這個指令能將資料夾掛在成虛擬磁碟機，
當你有好幾層的目錄又常常使用他的時候，
就可以利用該功能，直接在檔案總管看到他，
不用一層一層的點進去。

使用方法

如何掛在磁碟

subst 要掛在的磁碟代號 要掛在的資料夾路徑
假設我在 D: 底下有個目錄 a d:\a
我想把她掛載到 z 磁碟的話我應該這樣寫
subst z: d:\a

查看掛了哪幾個碟

subst
直接輸入就可以
他會列出你目前掛了哪一些碟

刪除掛在的硬碟

之前我們掛了 z
要刪掉他你要這樣寫

subst z: /D

使用場警

你可以將 onedrive 直接掛在到虛擬磁碟中
當然，onedrive 要被打開
不然他就只是個資料夾而已。

這是我 onedrive 的目錄路徑
D:\onedrive\3\OneDrive - 淡江大學視障資源中心

想要知道自己的 onedrive 存放路徑您可以
win + b 跳到工具列
找到 onedrive enter
tab 找到開啟資料夾 enter
F4 查看路徑 複製起來

輸入
subst z: onedrive_path
subst z: D:\onedrive\3\OneDrive - 淡江大學視障資源中心

每次都輸入也太麻煩

如果每次開機我都要這樣掛在
那不如不用
路徑也太長了吧！
沒有關係，只要這樣做就可以了

建立一個純文字文件
打開記事本在裡面輸入

@echo off
echo Function: assign folder to be a virtual disk
echo Author: mh
echo Start assign folder...
echo subst Z: "D:\onedrive\3\OneDrive - 淡江大學視障資源中心"
subst Z: "D:\onedrive\3\OneDrive - 淡江大學視障資源中心"
subst
echo Work is done at %time% %date%
echo All work Completed!
pause

用複製的就可以
記得路徑改移下
然後存檔
修改黨名成為 start_subst.bat

開機自動執行

我們將這個 .bat 複製貼上到啟動項目裡頭
首先找到檔案複製他
win + r 打開執行輸入框
輸入 shell:startup enter
貼上該 .bat 檔案

重新開機後 應該就會自動執行了
