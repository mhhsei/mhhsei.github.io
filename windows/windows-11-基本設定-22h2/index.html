Windows 11 基本設定 22H2

 windows  1780  4 分鐘

來寮 win 11 設定之前，
先來看看設定本身吧！

0. 設定的介面介紹

  • win + i 打開設定

此時會停在可編輯的區塊，若您知道設定的名稱，可以直接輸入關鍵字如注音，他會跟你說有幾個結果只要點擊他，就可以跳轉到該設定

  • 按下 tab or 下方向鍵會看到一個清單 可能會聽到 系統 11 之 1 第 1 級

這事告訴您選單目前焦點停在哪一項 一共有幾項的資訊

  • 按下 tab 會聽到 系統 1 之 1

這是個階層 之後會再介紹

  • 一直按下 tab 直到聽到 顯示器 15 之 1

這就是系統這個項目裡的內容
若跳到其他的項目如 個人化 那顯示的就不是這些內容

  • 用游標上下選擇 顯示器 並按下 enter

此時會聽到有關亮度的設定

你可以使用 tab 來瀏覽設定
若聽到設定 摺疊 這樣子的關鍵字 請用 [space] 展開併案下 tab 看看有那些設定

  • 我們利用 shift + tab 回到 系統 2 之 1

這邊可以用方向左右見選擇你要跳轉的層級
前提是你必須打開
這在你展開了許多項目的時候很有用
可以直接前往妳要去的設定葉面

1. 如何知道這台電腦的裝置規格、Windows 規格？

利用 cmd 命令

win + r 打開執行對話框
輸入 cmd
接著輸入 systeminfo
利用小鍵盤的 7,9 來閱讀資訊

設定

  • win + i
  • tab 找到系統並按下 enter
  • 多按許多嚇得 tab 找到清單 可能會寫顯示器 游標上下移動找到 系統資訊 15 之 15 類似的選項
  • 接著摁下 tab
  • 聽到 裝置規格 複製 這邊就可以利用 nvda + numpad(4,6) 來閱讀資訊
  • 在案 tab 聽到 windows 規格 複製 向上面一樣的方法也能閱讀資訊

tab and shift + tab 聽到的東西有一些不一樣要特別注意

2. 怎樣看有哪些輸入法？

  • win + i 打開設定面板
  • tab 找到系統游標往下找到 時間與語言 enter
  • 多按幾下 tab 找到 日期和時間 4 之 1
  • 游標往下 找到 語言與地區 4 之 2 enter
  • 此時會停在 windows 顯示語言 中文 (繁體，台灣) 折疊 這就是告訴你目前使用哪種語言
  • 按下 tab 找到 清單 中文 (繁體，台灣) 沒選取 2 之 1 類似的文字 按下 enter
  • tab 中文 (繁體，台灣) 群組 其他選項 enter
  • 游標上下可以移動 找到 語言選項 enter
  • 在案 tab 找到 清單 這一個清單就是告訴你有幾個輸入法
  • 剛剛經過的 新增鍵盤 打開後 就可以添加輸入法

3. 將微軟注音輸入法預設輸入模式社為英數與移除輸入法

接續上面的操作不再寫一遍

  • 在清單中找到 微軟注音 enter
  • tab 聽到 微軟注音 群組 鍵盤選項 enter
  • 這邊游標上下移動 移除就是移除輸入法 所以我們要點鍵盤選項
  • tab 找到一班 第一個選項就是了
  • 會停在夏拉方塊 上下可以選擇 英數與中文

使用舊版相容模式的輸入法

一直按下 tab 會聽到 使用舊版的微軟注音 按下

按下就代表著是使用舊版的輸入法 沒按下 就是新版囉！

4. 如何險示桌面圖示

  • win + i
  • tab 聽到系統接著游標往下找到 個人化 enter
  • 連續按下 tab 聽到 清單 佈景主題 enter 不是第一個清單 要住咿喔
  • 好 在按幾下 tab 聽到 桌面圖示設定 群組 開啟 按下 enter
  • 游標上下移動 會聽到一些選項 若有打勾 就會顯示到桌面 那麼利用 space 就可以決定是否要打勾
  • 選擇完成後 案 tab 找到套用 enter 在找到 確認 enter
  • 若想放棄更改 或不想改動 只要按下 esc 就可以了

5. 如何排序桌面捷徑

要使用舊版微軟輸入法才能這樣操作

  • win + d 回到桌面
  • ctrl + alt + space 聽到沒選取
  • 按下快顯功能鍵 游標往下找到 排序方式 enter
  • 這邊有一些項目 最重要的就是 名稱 項目類型這兩樣比較常用 決定要用哪種排序方式後直接按下關鍵的 enter 就設定完成了

若失敗直接在按下 F5 來重新整理看看

6. 如何排序桌面捷徑

  • win + i 按照上面的方法找到個人化 enter
  • 若您剛剛 佈景主題的設定沒有關掉 可以利用 tab or shift + tab 跳轉項目到 個人化 2 之 1 enter
  • 一樣利用 tab 找到 清單 背景這個清單中
  • 上下游標移動 找到 開始 enter
  • 利用 tab 可以瀏覽項目 建議將
    • 顯示最近新增的應用程式 按下
    • 顯示常用應用程式 按下

7. 如何指定預設應用程式？

  • win + i 一樣的方法找到 應用程式 enter
  • tab 找到 清單 已安裝的應用程式 此時游標往下找到 預設應用程式 enter
  • tab 找到清單 會看到許多的應用程式
  • 我們來設定瀏覽器的預設吧 以 brave 為例
    *. 在清單中找到 brave enter
    • 將 Brave 作為預設瀏覽器 enter
    • 會聽見 已設定您的預設瀏覽器 這樣子就可以設定完成

預設程式設定的方式還有其他的方式

8. 如何移除程式

  • 一樣是 win + i 找到應用程式 enter
  • tab 找到 清單 已安裝的城市 enter
  • tab 經過排序找到 清單
  • 游標上下找到要刪除的城市
  • 案移下 tab 並且 enter
  • 此時詹開的選單跟安裝程式有關係 所以有一些不太一樣 但沒關係 只要找到 解除安裝並 enter 就可以

有時候移除成事實 要回答一些問題 需要注意

9. 如何將 記事本 傳送到桌面當作捷徑

notepad 在 win 11 已經變成 app 的形狀了，
當你在開始功能表對他按又見 在更多里也沒有傳送到桌面的選項，
用 microsoft store 安裝的城市好像都這樣，
沒關係，那我們就把他揪出來再傳送就可以。

  • 打開畚箕進入 C,
  • 找到 windows 的資料夾，
  • 輸入 note 會看到一個 notepad.exe
  • 按下快險功能漸 找到 傳送到 enter
  • 游標上下找到 桌面 (建立捷徑) 桌 enter
  • 回到桌面看看 若對潔淨名稱不滿意在案 F2 修改名稱

記得要選當作潔淨的那個 否則就是複製 notepad.exe 到桌面喔
