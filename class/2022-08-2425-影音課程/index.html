如何進入 youtube

youtube

  1. 打開瀏覽器 firefox Chrome Brave 都行
  2. 抵達網址列 按下 alt + d
  3. 輸入 youtube > 現在的瀏覽器都有著搜尋的功能 尤其 youtube 算是熱門網站 所以可以這樣跳轉

播放操作

網頁上的按鈕一樣可以播放、快轉、倒轉、暫停等等，這不必多說，
我們要介紹的是更方便的操作方式。
那就先來找個影片來示範吧

  1. 游標左右建 倒轉與快轉 10s
  2. 數字鍵 0 - 9 0% to 90% 區間的跳轉見據為 10% > 建議直接使用大鍵盤上面的就可以 否則還需切換 num lock
  3. 空白建 暫停
  4. 游標上下鍵 大小聲的切換

  8. 條快速度 shift + .

  10. 跳往下一個章節 ctrl + right
  11. 靜音 m
  12. c 開關字幕 如影片提供字幕

搜尋

  • 首先確認自己的瀏覽器是在 youtube 的葉面
  • 確認 NVDA 在瀏覽模式 音效提示 [du]
  • 按下 ctrl + home 跳到葉守的地方
  • 按下英文字母 e 跳轉到搜尋框
  • 若跳轉過來沒有自動進入編輯模式可以按下 enter
  • NVDA 已經進入編輯模式那就可以開始輸入了 ex:最偉大的作品
  • 輸入 最偉大的作品 enter[2] 會跳轉到搜尋結果的葉面
  • 此時焦點會停在 搜尋篩選器 游標就可以往下瀏覽每個結果 且可以去讀取一下內容 可以試試看
  • 你會聽到結果的標題都是在標題層級三的位置
  • 所以可以使用 3 來快速的瀏覽 , shift + 3 反向瀏覽三級標題
  • 若結果符合期待 可以按下 enter 打開

若在搜尋框游標上下 可以找到最近搜尋的項目
firefox 較新的版本不會自動播放影片 可以到設定裡搜尋關鍵字 並將它改成自動

  • NVDA切換到焦點模式 NVDA + Space
  • 按下 c 若有聽到開啟或關閉的相關提示就代表有CC字幕

切換字幕語言與自動翻譯 如果有的話

  • 確認字幕已經開啟
  • ctrl + home

  • 游標上下找到 字幕 enter
  • 上下游標找尋自己想要的字幕語言 若有字幕會看到一個選單 enter 確定
  • esc 離開設定子功能錶

字幕選單最下面有翻譯 做法與上面的一樣

登入與登出

登入了youtube 有更多的設定可以使用，且可以提供觀看紀錄、收藏影片、上船影片等功能。

登入
  • ctrl + home 回到葉守
  • 按下 d 跳轉地標 找到導覽區 首頁
  • 游標往上找到登入的按鈕 enter
  • 分別輸入帳號與密碼並且勾選保持登入 tab 尋找登入
  • 若有兩步驟驗證，請掏出手機打開通知並點選是
登出
  • ctrl + home 回到葉守
  • 按下 d 跳轉地標 找到導覽區 首頁
  • 游標往上找到子功能表 點選個人資料相片，即可顯示您其他的帳戶 enter
  • NVDA 的焦點會停在主要內容區 地標 你的帳號名稱的地方
  • 游標往下找到登出 enter

可以注意NVDA朗讀物件的屬性，以方便日後可以快速跳轉

切換帳戶
  • 方法大致上與登出相同
  • 只是在找尋燈初選像之前，要先找到切換帳戶，目前的版本是放在登出的上方
  • 游標往下找到新增帳戶 enter
  • 接下來的步驟就等於登入另一個帳號
  • 登入了另一個帳號，就會以另一個帳號的身分使用youtube
  • 若想切換到原本的第一個帳號則必須在典籍點選個人資料相片，即可顯示您其他的帳戶 enter
  • 游標往下找到 切換帳戶 enter
  • 游標往下找到 其他帳戶
  • 繼續往下選擇您的帳號 並且 enter

設定

  • 游標往上找到子功能表 點選個人資料相片，即可顯示您其他的帳戶 enter

  • 游標往下找到設定 enter

  • 游標往下找到 播放與觀影體驗 enter

注意該播放設定只對應目前使用的瀏覽器，如果其他的瀏覽器您必須在設定一次包括手機也一樣

媒體庫的介紹

在這裡可以查看觀看紀錄、播放清單等項目

查看觀看紀錄
  • 利用單鍵瀏覽d 導覽區 地標 首頁 已選取
  • 游標往下找到 媒體庫 enter
  • 利用游標 往下找尋 主要內容區 地標 標題 第 2 級 到訪過 觀看記錄 或者利用數字2 跳轉標題
  • 利用大鍵盤的數字3可以查看結果 游標往下可以找到一些內容
  • 若要查看更多在觀看紀錄游標往下找到 該選項並按下enter 就可以顯示更多筆資料
播放清單
建立播放清單與新增影片到清單內
  1. 找到一個應片先別點進去 停留在標題層級三 結果的地方
  2. 游標往下 找到動作選單
  3. tab 找到 儲存至播放清單 enter
  4. tab 逛逛 會有幾個選項 取消 稍後觀看 建立新的播放清單 如果沒有建立過播放清單應該會長這樣
  5. 選擇 建立新的播放清單 如果已經有播放清單 而想將影片放在清單內 就選擇清單的名字enter
  6. 編輯清單名稱
  7. tab 會有個子功能表 可以調整隱私層級 公開 不公開 私人
  8. tab 找到建立

重新整理網頁 就可以看到新增的播放清單了
當然 你也可以建立一個空的播放清單 這邊就不掩飾了

如果是將一個影片加到建立好的清單

  • 找到影片後游標往下找到動作選單
  • 一樣的方式找到 儲存至播放清單
  • 勾選想要加入的清單
  • 按下 esc 離開

若在標題往下沒有找到動作選單 你就必須開啟影片 然後找到相關選項

從播放清單內移除影片與刪除播放清單

先來說說移除影片：

  • 找到播放清單
  • 游標往下找到 查看完整清單enter
  • 按下數字3 找到影片
  • 游標往下找到動作選單
  • tab 找到 從 xxx 播放清單中移除enter
  • 焦點會停在復原的地方 若後悔了 可以按下enter

接著，說說移除清單吧：

  • 找到播放清單後
  • 游標往下找到 查看完整播放清單
  • ctrl + home 回到葉守
  • 按下 d 到達主要內容區塊
  • 游標往下找到 動作選單enter
  • tab 找到 刪除播放餐單enter
  • 會有兩個選項 取消 刪除
  • 如果要刪除影片直接點選 刪除 就可以

download AudioControl-1.8.nvda-addon

使用方式：

  • 下一個聲音輸出設備 pc:NVDA+Windows+D nb:NVDA+Windows+D
  • 上一個聲音輸出設備 pc:Shift+NVDA+Windows+D nb:NVDA+Windows+D
  • 上一個應用程序/設備 pc:Windows+Control+Alt+小鍵盤4 nb:Windows+Control+Alt+Left
  • 下一個應用程序/設備 pc:Windows+Control+Alt+小鍵盤6 nb:Windows+Control+Alt+Right
  • 增大當前應用程序/設備的音量 pc:Windows+Control+Alt+小鍵盤8 nb:Windows+Control+Alt+Up

translation翻譯附加元件

這只有在新的NVDA 2019.3 以上才能使用

找不到就的附加元件

你只需要安裝他並且將翻譯打開，
就可以讓NVDA自動朗讀翻譯的語言，
雖然是機器翻譯，
但如果是要了解內容，
那大概是足夠的了。

download translate-2022.06.24.nvda-addon

你只要打開翻譯的功能，
預設的案件是 ctrl + shift + NVDA + t,
這會將資訊傳給 google 來做翻譯
並將翻譯的結果朗讀或用點字傳達給使用者，
必須注意的是它會傳送一切的內容，
所以，若有帳號、密碼、金融相關或隱私資訊，
請將附加元件關閉ctrl + shift + NVDA + t。

音訊與字幕選單

與其他家沒什麼不同

  • 音訊：就是選擇發音的語言 按照之前的設定 如果有中文 就以中文發音 沒有 則用影片原語言 當然你也可以自行調整 單指左右滑動選擇語言 點兩下確認就可以
  • 字幕選單：跟語言切換的方式一樣 而他有個選項是關閉 就是不產生字幕
  • 最後：四指點及螢幕下方 會有個關閉 這裡說的是 關閉字幕選單

播放操作

因為她播放式使用蘋果自家的播放器，
操作方式與在 safari 上是一樣的。

  • 典籍影片就可觀看
  • 雙指點兩下 可暫停 播放影片
  • 單指點兩下 影片 通常在螢幕中央 可以顯示播放控制向

顯示播放控制象的情況下

上方 由左至右

  • 關閉 典籍後 關閉播放器
  • 縮放 應該是調整螢幕
  • 子母畫面 就算回到主畫面 仍有小螢幕寬看
  • 靜音 決定是否靜音

下方選單：

  • 從影片的位置往下探索 會聽到 目前位置 單指上下波 可以調整時間軸10%的進度
  • 往右華 依序看到 經過時間 剩餘時間 更精確地了解時間
  • 往右華 airplay 高速藍芽傳輸
  • 往右華 倒轉 播放暫停 快轉 點選後 執行該功能

更多 語言、字幕、速度

  • 原本這些按鈕的狀態都是 以收合 當單指點兩下 可以沾開
  • 展開後 可以單指往右華 選擇語言或字幕或速度 當然也可用探索的方式
  • 決定了何種語言、字幕與速度後 單指點兩下 會跳回播放介面
  • 若將其中一項展開 再點選收闔 或展開另一個項目時 也會跳回播放介面

建議在調整十 可先按暫停 以聽清旁白的聲音
