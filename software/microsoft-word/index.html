2. 文档的新建保存与关闭等

建立檔案：word

1. win input -> 'word' 游標向下找到對應的城市
2. 桌面上通常有捷徑 那你就可以打開他
3. 打開了 word 按下 ctrl + n

剛打開了 word 會出現幾個項目
這個時候你直接選擇 空白文黨 就可以了
剛打開你可以使用 NVDA + t 閱讀一夏 他會跟你說 文件1 - word

存檔 ctrl + s or 另存新黨 F12
如果按下另存新黨 他會把第一航當作黨名
接著會打開對話框 其中按下 tab 可以瀏覽
值得注意的有：
副檔名的選擇 可以用上下游彪挑選一個副檔名 這決定了以何種方式存取檔案
儲存位置 與檔案總管相同的操作方式 預設是在文件裡頭
找到保存 按下 enter 就可以存檔 取消 就是不存檔
對於新建的檔案 ctrl + s 會使用另存新黨 而如果是已經存在的檔案則是存檔而已
存檔的時候要先選擇瀏覽的索引標籤 接著方式就跟另存一樣
關閉 word alt + F4 另外的方式以後再說
選擇取消 則會回到文黨編輯區
按下不保存 那就關掉了 word

以下是檔案總管當中樹狀檢視的操作方式 其他地方的樹狀檢視也是岔不多的原理
樹狀檢視可以顯示資料夾清單或電腦中的其他項目，資料夾在清單中會有「展開」或「折疊」的提示。在樹狀檢視的父項目展開後才能看到它的子資料夾或子項目，而在折疊的狀態下無法看見子項目。
例如 NVDA 的 documentation 資料夾被展開後，就可以使用下方向鍵移到裡面的各個語言子資料夾，而當被折疊時，使用下方向鍵會跳過這些子資料夾而移到與 documentation 平行的其他資料夾上。
使用右方向鍵可以展開被折疊的資料夾，但如果這個資料夾內沒有任何子資料夾的話，那就無法展開，也就是按右方向鍵後，什麼事都不會發生。
而已經被展開的資料夾，再按一次右方向鍵就會跳到展開後的第一個子資料夾上，這相當於下方向鍵的效果。
當資料夾被展開時，NVDA 會朗讀「展開」，並且告訴我們裡面有多少個項目(註一)。
使用左方向鍵可以折疊被展開的資料夾，若此資料夾本來就是折疊的，或者沒有任何子資料夾的話，那麼按左方向鍵會直接跳到它上一層級的父資料夾。
在點顯器上，折疊的項目會顯示「加號(+)」，展開的項目會顯示「減號(-)」。
如果該資料夾處於展開狀態，按一次下方向鍵會將焦點移到下一層級的第一個子資料夾上。
如果該資料夾處於折疊狀態或沒有任何子資料夾時，按一次下方向鍵會將焦點移到同一層級的下一個資料夾上。
在樹狀檢視的瀏覽中，NVDA 會朗讀每個項目的層級，最高層級是「第 0 級」，下個層級就是「第 1 級」、「第 2 級」等以此類推。在點顯器上層級的縮寫是「lv」，它會顯示在項目名稱的位置後面。

編輯連結

在連結的地方按下又見往下找到 編輯連結
若沒出現 則需要將連結選取起來

打開連結

按下又見 往下找到打開連結

刪除連結

選取後按下 del
這會全部刪掉
但若是想保留網址的方式

通常的情況下還是需要將連結選取才能做動作 編輯 刪除等等

若是想要把文件做成連結 絲路一樣 但案件不同 其操作方式就與當按總管選定檔案一樣
